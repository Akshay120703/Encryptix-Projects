Simple Steps to execute this project:
[NOTE:- Make sure flask is installed properly in your system. If not, please install flask as well as all the supported libraries.] 

1) Open the folder where flask application and all it's allied libraries and applications are stored.
2) Store all the above files in this folder.
3) Now, create a folder named "templates". Paste the Churn.html file in the "templates" folder. 
4) Open Command Prompt or any other terminal and navigate to the folder where the flask application and all your files are stored.
5) Type "flask run" in the terminal.

Open the port mentioned in the terminal and the project is executed!!
